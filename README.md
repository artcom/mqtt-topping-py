## Development

To install dev dependencies:

```sh
$ python3 -m venv venv
$ source venv/bin/activate
$ pip install -r requirements_dev.txt
```

To run unit tests:

```sh
$ python -m pytest tests
```

To release a new version:

Update the field `version` in `pyproject.toml` and tag the version.
